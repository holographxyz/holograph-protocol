# LayerZeroModuleV2 foundry scripts

Les scripts foundry `LayerZeroModuleV2` permettent de deployéer le nouveau module de message utilisant Layer zero V2 et de gérer les differentes fonctionalitée associer a ce nouveau module. Ce script est accessible à travers la commande suivante:
```bash
pnpm forge:layerZeroModuleV2
```

## Commandes disponibles

### `pnpm forge:layerZeroModuleV2`

Cette commande n'execute aucune fonctionalité ni aucune transaction, elle affiche simplement une aide basique sur les differents scriptes disponibles.

Elle permet aussi de visualiser quelles variables d'environnement sont détectées et pouvant être utilisées par les scripts.
